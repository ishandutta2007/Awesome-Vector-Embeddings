# 🕸️ Graph Embeddings

Graph embeddings convert nodes, edges, or entire network substructures into vectors.

## 🚀 Overview
This preserves the structural topology of the graph and powers modern recommendation engines (e.g., Node2Vec, GraphSAGE).
